== Description ==
Provides a customizable command line interface for performing data migrations from another DB into WP.

== Installation ==
1. Search for "DB2WP Import CLI" on the WordPress Plugin directory from your WordPress site dashboard.
2. Install the plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress.
